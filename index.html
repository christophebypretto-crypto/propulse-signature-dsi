Comment l'installer dans Gmail

  1. Clique sur « Copier la signature » ci-dessus.
  2. Dans Gmail, ouvre Paramètres ⚙️ → Voir tous les paramètres → onglet « Général » → section « Signature ».
  3. Clique sur « Créer » (ou modifie une signature existante) et colle avec Ctrl+V (Windows) ou Cmd+V (Mac).
  4. En bas, sélectionne-la dans « PAR DÉFAUT POUR LES NOUVEAUX E-MAILS », puis « Enregistrer les modifications ».
